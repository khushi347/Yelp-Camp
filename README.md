# 🏕️ YelpCamp

A full-stack campground discovery and review platform where users can explore, create, review, and manage campgrounds with secure authentication, interactive maps, and image uploads.

## 🚀 Features

* User authentication and authorization
* Create, edit, and delete campgrounds
* Upload campground images
* Interactive map integration
* Review and rating system
* Responsive user interface
* Secure route protection and validation

## 🧩 Problem Statement

Finding and sharing camping locations often requires information from multiple sources. Users need a centralized platform to discover campgrounds, share experiences, and manage listings.

## 💡 Solution

YelpCamp provides a community-driven platform where users can:

* Discover campgrounds
* Share reviews and ratings
* Upload images
* Manage campground listings
* Explore locations through integrated maps
